Braids Manager v2.0

Application Guide

PART I: GENERAL INFORMATION

1. The purpose of this application is to maintain a database of all the testing cables
2. Follow document ELECTRICAL TESTING PROCEDURE to see usage instructions

PART II: HOW TO USE

1. Download the application
2. Set all the settings according to PART V
3. Run App_Name.exe file

PART III: MAIN MENU

1. ADD NEW TEST CABLE
2. ADD
 2.1. This will allow you to add a new test cables to the database by filling a form
 2.2. At the end of the form press Y to submit the form and the new test cables will be added to the database
3. EDIT
 3.1. This will allow you to edit an existing test cable by filling a form
 3.2. At the end of the form press Y to submit the form and the test cable information will be edited
4. DELETE
 4.1. This will allow you to delete an existing barid only if it is the last one
